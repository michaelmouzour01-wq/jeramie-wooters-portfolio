# Jeramie Wooters - Professional Portfolio

A comprehensive portfolio showcasing expertise in electrical engineering, power systems analysis, and 7+ years of QGIS geospatial experience.

## Features

- **Technical Minimalism Design** with electric blue and navy color scheme
- **6 Featured Projects** with QGIS visualizations and technical diagrams
- **7-Year QGIS Timeline** showcasing progressive expertise (2017-2024)
- **Skills Matrix** displaying proficiency across 8 technical domains
- **Research Impact Metrics** highlighting 50+ publications and $2.03M in grants
- **Fully Responsive Design** optimized for all devices
- **Production-Ready** with optimized build and fast load times

## Technology Stack

- **Frontend**: React 18 + Vite
- **Styling**: Tailwind CSS 3
- **Build Tool**: Vite
- **Deployment**: Vercel

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm preview
```

## Deployment to Vercel

1. Push code to GitHub:
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/jeramie-wooters-portfolio.git
git push -u origin main
```

2. Deploy to Vercel:
   - Go to https://vercel.com/dashboard
   - Click "Add New" → "Project"
   - Import your GitHub repository
   - Click "Deploy"

3. Your portfolio will be live at `https://jeramie-wooters-portfolio.vercel.app`
